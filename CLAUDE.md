# CLAUDE.md

このリポジトリは dotfiles (設定ファイル群) を管理している。

## インストールスクリプトの同期ルール

新しい機能やツールが必要な改修を行った場合は、必ず `install.sh` (macOS) と `install-server.sh` (Linux/Ubuntu) の両方に反映すること。
これらのスクリプト単体の実行だけで、設定の上書きと必要ツールのインストールが完結する状態を維持する。

具体的には:
- 新しいツールを導入した場合 → 両スクリプトにインストール手順を追加
- 新しい設定ファイルを追加した場合 → 両スクリプトにシンボリックリンク作成を追加
- 新しい bin スクリプトを追加した場合 → 両スクリプトに ln -sf を追加
- プラグインマネージャ等の依存を追加した場合 → 両スクリプトに clone/install を追加
